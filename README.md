# SimonBedard

A repository that understands how Simon Bedard works, thinks, and excels. It holds
a clear, structured picture of who he is, what he has built, and what he is working
on, and it is meant to evolve continuously.

It is a map, not a warehouse. The heavy assets (the book, podcast episodes,
marketing content, the strategy model) live in their own repos. This repo
describes them and points to where they live, so anyone, or any AI, working here
can understand Simon quickly.

## Contents
- `profile.md` — who Simon is, how he works, and how he thinks.
- `body-of-work.md` — what he has built, with pointers to the full assets.
- `current-focus.md` — the living dashboard of active work.
- `CLAUDE.md` — how an AI session should use this repo.

## How it evolves
Update `current-focus.md` as work shifts, and add to `body-of-work.md` when
something new is finished. Never invent facts about Simon; mark unknowns
"(to confirm)".
